# Métodos Numéricos · Teoría y Calculadora

Aplicación web interactiva para el aprendizaje de métodos numéricos. Incluye:

- 📖 **Teoría** con explicaciones, fórmulas (MathJax) y ejemplos.
- 🧮 **Calculadora** con selector de métodos, paso a paso y resultados.

## Métodos disponibles

- Errores numéricos (absoluto, relativo, porcentual)
- Bisección
- Newton-Raphson
- Jacobi (sistemas lineales)
- Integración (Trapecio, Simpson)
- Diferenciación (progresiva, regresiva, centrada)

## Cómo usar

1. Abre `index.html` en tu navegador.
2. Elige entre **Teoría** o **Calculadora**.
3. En la calculadora, selecciona un método, ingresa los datos y presiona **Calcular**.

## Tecnologías

- HTML5, CSS3, JavaScript
- MathJax para fórmulas matemáticas
- Modo oscuro/claro con persistencia
